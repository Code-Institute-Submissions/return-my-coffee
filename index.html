Return myCoffee

While you are here, you have a task do to. It's all very terrible, but I must tell you that your cup of coffee has been stolen. By whom, I'm afraid I don't know. The only way you can hope to get it back is by completing each challenge within this game, and if you're lucky, you might have your coffee in your grasp once again!

Now you must want to get to it, then - there's no time to waste!

Perhaps you are curious about this game's creator? Well, curiousity is welcomed! Look below to read more about it.

This project started as a way for me to test and practice my newly acquired JavaScript skills, and see how much I could push myself. After completing one game I decided to do another, then another. It was then when I decided to combine all of these games together. To do that successfully, however, I needed something to tie it all together - which is how Return myCoffee took shape.
